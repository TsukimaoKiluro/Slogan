# AI 智能体学习辅助应用 - 技术文档

## 项目概述

本项目是一个沉浸式 AI 智能体学习辅助应用，通过人工智能技术为用户提供个性化的学习支持。应用采用现代化的单页面设计，支持 Markdown 编辑、PPT 生成、错题管理等多种学习工具。

### 技术栈

- **前端框架**: Vue 3 + Vite
- **UI 组件**: daisyUI 5 + Tailwind CSS
- **后端框架**: Node.js + Express
- **数据库**: SQLite
- **PPT 生成**: PptxGenJS

---

## 核心功能模块

### 1. AI 助手 (`/`)

智能对话助手，支持多轮对话、拍照搜题、资料生成：
- **AI 对话**：支持流式输出的智能对话
- **内容生成**：AI 可自动识别并生成以下内容：
  - **PPT 演示文稿**：自动生成真正的 PPT 文件，保存到 `uploads/ppt/` 并同步到学习中心
  - **Markdown 笔记**：生成结构化的 Markdown 笔记，支持 LaTeX 公式
  - **视频脚本**：生成带时间轴的视频脚本
- **Action 卡片**：AI 回复中的生成卡片，点击"生成"按钮可一键生成对应内容
- **自动导入**：生成的内容自动导入学习中心，可在学习中心查看和管理

### 2. 学习中心 (`/learning`)

学习资料管理中心，支持：
- 按学科分类管理资料
- 知识点关联与检索
- 学习进度追踪

### 3. 个人中心 (`/profile`)

用户画像与学习数据分析：
- 知识基础评估
- 认知风格分析
- 易错点记录
- 学习目标设定
- 兴趣方向分析
- 学习记录统计

### 4. Markdown 编辑器 (`/markdown`)

支持实时预览、LaTeX 公式、代码高亮的 Markdown 编辑器。

### 5. PPT 阅览器 (`/ppt-viewer`)

在线预览 PPT 文件，无需下载即可查看：
- **幻灯片导航**：支持上一页/下一页、跳转到指定页
- **缩放控制**：支持放大/缩小/适应屏幕
- **全屏模式**：支持 F11 全屏浏览
- **缩略图预览**：底部显示幻灯片缩略图列表
- **键盘快捷键**：←→键翻页，+-缩放，Home/End跳转首尾
- **文件下载**：支持直接下载 PPT 文件
- **自动解析**：使用 JSZip 解析 PPTX 文件结构

### 5. PPT 生成器 (`/ppt-editor`)

通过简单的大纲编辑，自动生成专业的 PPT 演示文稿：
- **预设多种模板**：包含 12 套学科专用模板（高等数学、线性代数、概率论、大学物理、专业课、英语、编程、政治、知识梳理、小组展示、毕业答辩、培训课件）
- **学科主题风格**：学术蓝调、数学紫韵、物理绿光、英伦金风、编程深空、政治红韵等 6 种风格
- **配色方案**：蓝色系、紫色系、绿色系、红色系、金色系
- **用户模板管理**：可保存、删除自定义模板
- **实时预览**：边编辑边预览效果
- **一键生成**：自动保存到 `uploads/ppt/` 目录并同步到学习中心
- **同步保存**：生成后自动保存到学习中心的 learning_materials 表
- **任务追踪**：生成过程有 AI 任务进度追踪

### 6. PPT 阅览器 (`/ppt-viewer`)

在线预览 PPT 文件，无需下载即可查看：
- **幻灯片导航**：支持上一页/下一页、跳转到指定页
- **缩放控制**：支持放大/缩小/适应屏幕
- **全屏模式**：支持 F11 全屏浏览
- **缩略图预览**：底部显示幻灯片缩略图列表
- **键盘快捷键**：←→键翻页，+-缩放，Home/End跳转首尾
- **文件下载**：支持直接下载 PPT 文件
- **自动解析**：使用 JSZip 解析 PPTX 文件结构

### 7. 工作进度追踪面板

悬浮在页面左下角的抽拉式气泡面板：
- **收缩状态**：显示圆环进度条，实时显示最新任务进度
- **展开状态**：显示详细任务列表，包含进度条和状态
- **自动同步**：与后端 AI 任务 API 实时同步
- **浏览器标签样式**：类似浏览器下载管理器

### 8. 工具中心 (`/tools`)

聚合多种学习工具的入口页面。

### 学科自动识别机制

系统支持根据内容自动识别学科：

1. **匹配优先级**：
   - 完全匹配（display_name 或 name）：100分
   - 包含匹配（标题包含学科名）：80分
   - 关键词匹配：60-90分

2. **自动创建**：未匹配到时自动创建新学科

3. **调用场景**：
   - AI 生成 PPT/MD/Video 时
   - 上传学习资料时

### 掌握状态与复习间隔

| 状态 | 标识 | 复习间隔 | 说明 |
|-----|------|---------|------|
| 待掌握 | `not_mastered` | 1天 | 新增或复习后仍错误的错题 |
| 掌握中 | `learning` | 3天 | 正在学习的错题 |
| 已掌握 | `mastered` | 7天 | 多次复习后已掌握的错题 |

### 错误原因分类

| 原因 | 标识 | 说明 |
|-----|------|------|
| 概念不清 | `concept` | 对知识点理解不透彻 |
| 计算错误 | `miscalculation` | 计算过程中的失误 |
| 粗心大意 | `carelessness` | 审题正确但做错 |
| 审题不清 | `misread` | 理解题意有偏差 |
| 其他 | `other` | 其他原因 |

## 题目库管理

题目库使用 JSON 文件存储 (`data/exam_questions.json`)，支持按学科分类管理，方便题目复用。

### 题目数据结构

```json
{
  "id": "math_001",
  "type": "single",           // 题型: single/multiple/fill/judge/comprehensive
  "difficulty": "medium",     // 难度: easy/medium/hard
  "knowledgePoints": ["极限", "连续性"],
  "content": "题目内容（支持LaTeX）",
  "options": ["A. 选项1", "B. 选项2", "C. 选项3", "D. 选项4"],
  "answer": "C",              // 答案
  "score": 5,                 // 分值
  "analysis": "解析内容"       // 答案解析
}
```

### 题目库 API

```
GET    /api/question-bank/subjects                    - 获取学科列表
GET    /api/question-bank/:subject/questions          - 获取学科题目列表
GET    /api/question-bank/:subject/stats              - 获取学科统计
POST   /api/question-bank/:subject/questions          - 添加题目
PUT    /api/question-bank/:subject/questions/:id      - 更新题目
DELETE /api/question-bank/:subject/questions/:id      - 删除题目
```

### 预置题目

目前预置了以下学科的题目：
- **高等数学** (math)：极限、导数、积分、级数、多元函数等
- **线性代数** (linear_algebra)：矩阵、行列式、特征值、向量组等
- **概率论** (probability)：概率、条件概率、随机变量、期望、正态分布等
- **大学物理** (physics)：力学、电磁学、能量守恒、热力学等

### 生成方式

1. **从题库生成**：从本地题库中随机抽取题目，生成速度快，题目质量稳定
2. **AI智能生成**：由AI根据设置生成新题目，题目更灵活，可定制性强
