# 题目库使用指南

## 概述

题目库使用 JSON 文件存储 (`data/exam_questions.json`)，支持按学科分类管理题目，方便重复使用。

## 题目库结构

```
data/exam_questions.json
├── description: 题目库描述
├── version: 版本号
├── lastUpdated: 最后更新时间
├── subjects: 学科分类
│   ├── math: 高等数学
│   ├── linear_algebra: 线性代数
│   ├── probability: 概率论
│   └── physics: 大学物理
└── comprehensive: 综合应用题模板
```

## 题目格式

### 单选题 (single)

```json
{
  "id": "math_001",
  "type": "single",
  "difficulty": "medium",
  "knowledgePoints": ["极限", "连续性"],
  "content": "题目内容（支持LaTeX: $x^2$）",
  "options": ["A. 选项1", "B. 选项2", "C. 选项3", "D. 选项4"],
  "answer": "C",
  "score": 5,
  "analysis": "答案解析"
}
```

### 多选题 (multiple)

```json
{
  "id": "math_005",
  "type": "multiple",
  "difficulty": "hard",
  "knowledgePoints": ["多元函数", "偏导数"],
  "content": "多选题内容",
  "options": ["A. 选项1", "B. 选项2", "C. 选项3", "D. 选项4"],
  "answer": ["A", "B", "C"],
  "score": 10,
  "analysis": "解析"
}
```

### 填空题 (fill)

```json
{
  "id": "math_003",
  "type": "fill",
  "difficulty": "easy",
  "knowledgePoints": ["积分"],
  "content": "$\\int_0^1 x^2 dx = $ ______",
  "answer": "\\frac{1}{3}",
  "score": 5,
  "analysis": "解析"
}
```

### 判断题 (judge)

```json
{
  "id": "math_004",
  "type": "judge",
  "difficulty": "medium",
  "knowledgePoints": ["级数", "收敛性"],
  "content": "若级数收敛，则通项趋于0",
  "answer": true,
  "score": 5,
  "analysis": "解析"
}
```

### 综合应用题 (comprehensive)

```json
{
  "id": "comp_001",
  "type": "comprehensive",
  "difficulty": "hard",
  "knowledgePoints": ["综合应用"],
  "content": "题目描述",
  "subQuestions": ["（1）...", "（2）...", "（3）..."],
  "referenceAnswer": "参考答案要点",
  "score": 20,
  "analysis": "解析"
}
```

## 使用方式

### 1. 从题库生成考试

在学习中心点击"生成阶段性测试"，选择：
- **生成方式**: "从题库生成"
- **学科**: 选择已有题目的学科
- **题目数量**: 设置各题型数量
- **难度**: 选择题目难度

系统会自动从题库中随机抽取符合条件的题目。

### 2. AI智能生成

选择"AI智能生成"模式，系统会调用配置的AI服务生成新题目。

## API接口

### 获取学科列表

```bash
GET /api/question-bank/subjects
```

### 获取学科题目

```bash
GET /api/question-bank/:subject/questions?type=single&difficulty=medium&limit=10
```

参数：
- `type`: 题型筛选
- `difficulty`: 难度筛选
- `limit`: 限制数量

### 获取学科统计

```bash
GET /api/question-bank/:subject/stats
```

返回：
```json
{
  "total": 20,
  "byType": { "single": 10, "multiple": 5, "fill": 3, "judge": 2 },
  "byDifficulty": { "easy": 5, "medium": 10, "hard": 5 }
}
```

### 添加题目

```bash
POST /api/question-bank/:subject/questions
Content-Type: application/json

{
  "type": "single",
  "difficulty": "medium",
  "content": "题目内容",
  "options": ["A", "B", "C", "D"],
  "answer": "A",
  "score": 5,
  "analysis": "解析"
}
```

### 更新题目

```bash
PUT /api/question-bank/:subject/questions/:questionId
```

### 删除题目

```bash
DELETE /api/question-bank/:subject/questions/:questionId
```

## 扩展题库

### 手动编辑JSON文件

直接编辑 `data/exam_questions.json` 文件，按格式添加新题目。

### 通过API添加

使用上述API接口添加题目，系统会自动保存到JSON文件。

### 批量导入

可编写脚本读取其他格式的题目数据，转换为JSON格式后导入。

## 考试测试页面

访问路径：`/exam`

### 功能特性

1. **多种题型支持**
   - 单选题：单选按钮选择
   - 多选题：复选框多选
   - 填空题：文本输入
   - 判断题：正确/错误选择
   - 综合应用题：多小问+文本作答

2. **考试模式**
   - **考试模式**：提交后统一批改，显示成绩和解析
   - **练习模式**：做一题给一题答案，即时反馈

3. **倒计时功能**
   - 显示剩余时间
   - 少于5分钟时变红警告
   - 时间到自动提交

4. **答题卡**
   - 快速跳转到指定题目
   - 显示已答/未答状态

5. **成绩统计**
   - 得分、正确率统计
   - 正确/错误题数
   - 答案解析查看

### 使用流程

1. 在学习中心点击"生成阶段性测试"
2. 选择生成方式（题库/AI）、学科、题目数量等
3. 点击生成，自动跳转到考试页面
4. 作答完成后点击"提交试卷"
5. 查看成绩和解析

## 注意事项

1. **ID唯一性**：题目ID在同一学科内必须唯一
2. **LaTeX支持**：题目内容支持LaTeX公式，使用 `$...$` 或 `$$...$$` 包裹
3. **知识点标签**：建议为题目添加知识点标签，便于按知识点筛选
4. **难度分级**：easy（简单）、medium（中等）、hard（困难）
5. **备份**：定期备份 `data/exam_questions.json` 文件
