## 未使用字段说明

### 未使用字段列表

1. **各表 id 主键**（除 study_goals 表外）
2. **各表 created_at 创建时间**

### 优化建议

- **`created_at` 字段**：
  - 可用于按创建时间排序查询
  - 可用于筛选特定时间段内的记录
  - 可用于实现"最近学习记录"等功能

- **各表 `id` 主键**：
  - 可作为 API 返回值的唯一标识符
  - 可用于构建资源的详细访问 URL
  - 可用于前端列表渲染时的 key 属性

## 学科管理模块（v1.0.1 新增）

### subjects 表结构

| 字段 | 类型 | 说明 |
|-----|------|------|
| name | TEXT | 学科标识（唯一键） |
| display_name | TEXT | 显示名称 |
| icon | TEXT | 图标标识 |
| color | TEXT | 主题颜色 |
| sort_order | INTEGER | 排序权重 |

### API 接口

| 方法 | 路径 | 说明 |
|-----|------|------|
| GET | /api/subjects | 获取学科列表 |
| POST | /api/subjects | 添加学科 |
| PUT | /api/subjects/:name | 更新学科 |
| DELETE | /api/subjects/:name | 删除学科 |

### 功能说明

1. **学科关联**：knowledge_base 等表的 subject 字段关联 subjects.name
2. **级联初始化**：添加学科时自动创建 knowledge_base 对应数据
3. **级联删除**：删除学科时自动清理相关数据
4. **动态显示**：前端雷达图等组件根据 subjects 表动态渲染
